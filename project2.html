Project #2: CellPyAbility - High-Throughput Dose-Response Screening

GitHub Repo

Stack: Python 3 • CellProfiler • PyInstaller • SciPy • Plotly • GitHub Actions

Objective: Improve cell viability analysis by automating high-throughput nuclei counting and statistical modeling. Open-sourced and available as a code-free Windows application.

Featured in Yale School of Medicine News.

Nuclei Counting is Superior to Metabolic Proxies

Standard high-throughput screenings often rely on metabolic proxies (e.g. tetrazolium reduction or ATP content) to estimate cell death. While fast, these methods are often confounded by metabolic variability or redox-active drugs. Further, they lack cell-level resolution, giving a single fluorescent value per well that often loses linearity.

Direct nuclei counting offers a ground-truth measurement of survival with single-cell resolution and higher accuracy. However, the computational cost of analyzing high-content images typically forces researchers back to inferior metabolic proxies. CellPyAbility eliminates this trade-off, processing 60 whole-well images in under one minute on commodity hardware.

Software Architecture and Accessibility

To ensure accessibility across the technical spectrum, I engineered a dual-architecture distribution:

  • For Computational Biologists: A modular Python package (available via PyPI and Bioconda) integrating CellProfiler for headless image segmentation and NumPy/Pandas/SciPy for statistical analysis.
  • For Bench Scientists: A standalone, code-free Windows executable (built with PyInstaller) that bundles the Python runtime and dependencies, requiring no distribution or coding knowledge.

Mathematical Implementation

The suite consists of two core analytical modules designed for robust statistical inference:

  • Logistic Regression: The GDA module fits dose-response curves using a 5-parameter logistic (5PL) model, then solves for the IC50 for each condition. Depending on the experiment, the ratio of IC50s can be used as an approximate therapeutic index.
  • Synergy Modeling: The Synergy module models 59 unique drug interactions per experiment. It calculates Bliss Independence scores and renders interactive 3D surface maps using Plotly to visualize synergistic volumes.
